대출 한도 계산

DSR 규제와 LTV를 모두 반영한 정확한 대출 한도를 자동 산출합니다.

DSR 한도

원천징수 총급여(계약연봉+인센) 기준 DSR 40% 적용. 스트레스 금리 3% 가산. 30년 원리금균등 기준 연간 원리금 상환액이 연소득의 40% 이내.

LTV 한도

생애최초 70% / 무주택 50% / 1주택 이하 적용. 9억 이하, 9~15억 구간별 차등 적용. 15억 초과 시 대출 제한.

최종 한도

DSR 한도와 LTV 한도 중 작은 값이 실제 대출 가능 금액. 구간별 최적 대출액 자동 계산.

기존 플랫폼과의 차이

단순 데이터 뷰어가 아닌, "의사결정 에이전트"를 지향합니다.

내집마련 AI 비서

  • 내 재무 상황(연봉/종잣돈/지출) 입력 → 맞춤 추천
  • DSR + LTV + 스트레스 금리 자동 계산
  • 스코어링 기반 TOP N 랭킹 (6개 항목 종합)
  • 전세가율 기반 갭투자 모드
  • 토허제(10.15) 전후 가격 변동 비교
  • 회복률 기반 저평가 발굴
  • 시기별 비교 (상승기 고점 → 하락기 저점 → 현재)
  • 연도별 매수 로드맵
  • 월 상환 시뮬레이션 (3가지 시나리오)
  • 100% 합법 — 공공 API만 사용

기존 부동산 플랫폼

  • 검색 → 매물 목록 → 직접 비교 (데이터 뷰어)
  • 대출 한도는 별도 계산기 필요
  • 랭킹/점수 없음 — 스스로 판단 필요
  • 전세가율 직접 계산해야 함
  • 정책 영향 분석 없음
  • 시세 변동은 보여주지만 "저평가"는 안 알려줌
  • 과거 시기별 비교 어려움
  • 장기 플랜 기능 없음
  • 대출 시뮬레이션 기본적 수준
  • 크롤링 기반 (법적 리스크 존재)
⚠️ 본 서비스는 참고용 정보 제공 도구입니다. 투자 판단은 본인 책임이며, 실거래가는 시점·층·조건에 따라 다를 수 있습니다. 대출 한도는 은행/신용등급/상품에 따라 달라지므로 반드시 은행 상담을 병행하세요.
