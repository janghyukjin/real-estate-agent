대출 한도 계산

DSR 규제와 LTV를 모두 반영한 정확한 대출 한도를 자동 산출합니다.

DSR (총부채원리금상환비율)

연간 원리금 상환액이 연소득의 40%를 넘을 수 없습니다.

기준 소득

원천징수 총급여 (계약연봉 + 인센티브/상여). 세전 금액 기준. 연봉이 높을수록 대출 한도 증가.

DSR 40%

연간 원리금 상환액 ≤ 연소득 × 40%. 모든 부채(주담대 + 기타) 합산. 집피티에서는 주담대만 계산.

스트레스 금리

실제 금리 + 3%p 가산하여 DSR 심사. 금리 4% → 심사 시 7% 적용. 2025-10-15 대책으로 1.5% → 3.0% 상향.

7월 3단계 (2025-07)

주담대 + 신용 + 기타 + 1주택자 전세대출 모두 DSR 합산. 한도 약 10~15% 축소.

예시: 연봉 5,000만원 / 금리 4%

연간 상환 한도 5,000만 × 40% = 2,000만원/년
스트레스 심사 금리 4% + 3% = 7%
30년 원리금균등 기준 DSR 한도 약 2.5억

LTV (담보인정비율)

10.15 대책 (2025-10-15) 반영. 매수자 유형 × 시가별 한도 둘 다 적용.

매수자 유형 LTV % 15억 이하 cap 15~25억 cap 25억 초과 cap
생애최초 70% 6억 4억 2억
무주택 40% 6억 4억 2억
처분조건부 1주택 (6.27) 40% 6억 4억 2억
1주택 (유지) 0% 대출 불가
다주택 0% 대출 불가

예시: 생애최초 / 18억 아파트

LTV 70% 18억 × 70% = 12.6억
15~25억 cap 4억 (binding)
최종 대출 4억 (cap 적용)

토허제 갭매수 (2026-05-12 신규)

세 낀 매물 매수 시 입주 유예 + 주담대 가능. 단 전세가율 40% 룰 적용.

실거주 유예

임대차계약 종료까지, 최대 2028-05-11까지 입주 유예. 잔금일 후 6개월 또는 임대종료+1개월 중 늦은 시점까지 전입신고 의무 면제.

매수자 자격

발표일(2026-05-12)부터 계속 무주택 유지. 발표 이후 처분으로 무주택된 자는 제외. 토허 신청 마감 2026-12-31.

전세가율 40% 룰

인수 임차보증금이 매수가의 40% 초과 시 대출 불가. 무주택자 LTV 40% (최대 6억) 기준 적용. 자기자본으로 충당해야.

의무거주 2년

본인 입주 후 2년 의무거주. 예: 2028-05 입주 → 2030-05까지 거주. 위반 시 토허법 처벌 (3년 이하 징역 or 3,000만원 벌금).

예시: 생애최초 / 매수 18억 + 인수 전세 7억

전세가율 7억 / 18억 = 38.9% ✓ (40% 이하)
LTV 한도 18억 × 40% (정책 적용) = 7.2억
대출 가능 min(7.2억, cap 4억) = 4억
자기자본 필요 18 - 7 - 4 = 7억

예시 (룰 위반): 매수 19.5억 + 인수 전세 8.6억

전세가율 8.6 / 19.5 = 44.1% ✗ (40% 초과)
대출 가능 0 (대출 불가)
자기자본 필요 19.5 - 8.6 = 10.9억 (전액)

최종 대출 한도

DSR 한도와 LTV 한도 중 작은 값이 실제 대출 가능 금액입니다.

구간별 최적화

집값이 올라갈수록 LTV 비율이 내려가므로, 단순히 "집값 × LTV%"가 아닌 구간별로 최적의 대출액을 역산합니다. 종잣돈이 많을 때 더 높은 가격대의 집을 살 수 있는 경우도 자동 계산.

매수 가능 집값

종잣돈 + 최종 대출 한도 = 매수 가능 집값. 이 범위(±10~20%) 내의 아파트를 스코어링하여 추천합니다.
