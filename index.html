기존 플랫폼은 데이터를 보여주고, 집피티는 판단을 도와줍니다.

집피티 — 스킬 기반 에이전트

  • 내 재무 상황 입력 → 원클릭 전략 스킬로 맞춤 추천
  • 저평가 발굴 / 소액갭 / 학군 / 급매 — 목적별 스킬
  • 스킬 조합으로 나만의 투자 기준 생성 (곧 출시)
  • DSR + LTV + 스트레스 금리 자동 계산
  • 6개 항목 종합 스코어링 → TOP N 랭킹
  • 22년 고점 대비 회복률 → 저평가 자동 태깅
  • 토허제(10.15) 전후 가격 변동 비교
  • 연도별 매수 로드맵 + 월 상환 시뮬레이션
  • 100% 합법 — 공공 API만 사용

기존 부동산 플랫폼

  • 검색 → 매물 목록 → 직접 비교 (데이터 뷰어)
  • 모든 사용자에게 같은 결과
  • 나만의 전략 저장/재사용 불가
  • 대출 한도는 별도 계산기 필요
  • 랭킹/점수 없음 — 스스로 판단 필요
  • 시세 변동은 보여주지만 "저평가"는 안 알려줌
  • 정책 영향 분석 없음
  • 장기 플랜 기능 없음
  • 크롤링 기반 (법적 리스크 존재)
⚠️ 본 서비스는 참고용 정보 제공 도구입니다. 투자 판단은 본인 책임이며, 실거래가는 시점·층·조건에 따라 다를 수 있습니다. 대출 한도는 은행/신용등급/상품에 따라 달라지므로 반드시 은행 상담을 병행하세요.
