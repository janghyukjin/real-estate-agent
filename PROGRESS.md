# 부동산 AI 비서 — 개발 진행 기록

## 2026-03-09 (Day 1) 설계 & 개발 완료 내역

### 프로젝트 개요
- **컨셉**: 무주택 2030 대상 "내집마련 AI 비서"
- **차별점**: 데이터 뷰어(리치고/호갱노노)가 아닌 "의사결정 에이전트"
  - 재무 분석 + 맞춤 추천 + 연도별 로드맵 + 대출 상환 시뮬레이션
- **기술 스택**: Python + Claude API (tool-use) + 공공 API 3종 + Kakao API

### 완성된 모듈

#### 1. 자금 계산기 (`src/calculator.py`)
- DSR/LTV 계산 (10.15 부동산 대책 반영)
- 스트레스 금리 3% 가산, 시가별 대출 한도 (15억↓ 6억 / 15~25억 4억 / 25억↑ 2억)
- 생애최초/무주택/1주택/다주택 유형별 계산
- 연도별 매수 가능 금액 로드맵
- 대출 상환 시뮬레이션 (원리금/거치식/조기상환)

#### 2. 실거래가 API 클라이언트 (`src/api_client.py`)
- 국토교통부 아파트 매매 실거래 상세 API ✅ 연동 완료
- 국토교통부 아파트 전월세 실거래 API ✅ 연동 완료
- 영문/한글 필드명 모두 지원 (API 버전 호환)
- 서울 25개구 + 경기 주요 시군구 코드 내장
- 서울 지역 등급 분류 (상급지/중상급지/중하급지/하급지)

#### 3. 건축물대장 API 클라이언트 (`src/building_ledger.py`)
- 국토교통부 건축물대장 총괄표제부 API ✅ 연동 완료
- 서울 전역 스캔 → 1,084개 아파트 세대수 캐시 (`data/apt_cache.json`)
- 300세대 이상 대단지: 664개
- 실시간 세대수 조회 + 캐시 업데이트 기능

#### 4. 갭투자 분석 (`src/kb_client.py`)
- 매매가 vs 전세가 매칭 → 전세가율 계산
- 갭투자 등급 산정 (적정/자본금필요/부적합/역전세위험)
- 지역 전체 갭분석 일괄 실행

#### 5. 카카오맵 연동 (`src/enrichment.py`)
- 지하철역/학교/편의시설 검색 (준비 완료, 미연결)

#### 6. 알림 시스템 (`src/alerts.py`)
- 관심 아파트 워치리스트 관리
- 신규 거래 감지 알림 기능

#### 7. Claude AI 에이전트 (`src/agent.py`)
- Claude Sonnet tool-use 기반 대화형 에이전트
- 8개 도구: 예산계산, 실거래검색, 추이분석, 지역정보, 갭분석, KB시세, 워치 등
- 대화형 CLI (`src/cli.py`)
